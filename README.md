# Number row characters

* ! - Exclamation mark, Factorial
* @ - At symbol
* \# - Hash, Pound
* $ - Dollar sign, Cash
* % - Percent, Percentage
* ^ - Caret, Exponent
* & - And, Ampersand
* \* - Asterisk, Star
* ( - Left parenthesis, Open parenthesis
* ) - Right parenthesis, Close parenthesis
